# Agent NFT Intelligence

Curated market exhibition for NFT collection intelligence, rarity signals, liquidity, and collector behavior.

## Live Demo

Static demo deployed through GitHub Pages and prepared for multi-platform mirroring.

## What it demonstrates

- Distinct visual metaphor and interaction model
- Browser-only deterministic agent simulation
- Reviewer-friendly single-page product demo
- No backend dependency, secrets, wallet keys, or private credentials

## Stack

- HTML/CSS/JavaScript
- Static hosting compatible with GitHub Pages, Surge, Vercel, Netlify, Cloudflare Pages, and Render

## Safety

This repository contains only static demo code. No private API keys, wallet material, cookies, or account credentials are required.
